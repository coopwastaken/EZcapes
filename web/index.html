Install Instructions
1 Download the .zip file
2 Extract the folder inside
3 Drag into C:\XboxGames\Minecraft for Windows\Content\data\skin_packs\
4 Launch Minecraft → Classic Skins menu
Troubleshooting
Not showing up? You may need to remove any other custom skin packs you've installed. Delete all folders in:
C:\Users\{User}\AppData\Roaming\Minecraft Bedrock\Users\{Numbers or shared}\games\com.mojang\skin_packs
